Marble Madness Game Design Intro Project

About Marble Madness

Marble madness is a simple game where the goal is to collect as many gems as possible before all marbles are destroyed. Marbles move on their own, but the player controls the direction of the turn tiles and can speed up the marbles.

This game was entirely coded by me as an introductory project for my senior Game Design elective.

Note - this game is built using Unity Web Player and will not run in Google Chrome. Use Firefox or Safari to play the game. Source code can be viewed here.

Game Details

  • Click on turn tiles to rotate them
  • Click on marbles to activate a short speed boost
  • When marbles collide, both will lose health.
  • there are 3 power ups that can be picked up - health (red), a shield (blue), and a turbo boost (orange).
